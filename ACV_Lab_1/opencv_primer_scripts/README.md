# OpenCV Primer Scripts

Beginner-friendly OpenCV examples demonstrating fundamental image and video processing operations.

## Scripts

- **read_image.py** - Load and display images, access pixel values and image properties
- **pixel_values.py** - Work with individual pixel data and image dimensions
- **save_image.py** - Save processed images to disk
- **to_grayscale.py** - Convert color images to grayscale
- **draw_shapes.py** - Draw rectangles, circles, and other shapes on images
- **read_video.py** - Read and process video files frame-by-frame
- **read_webcam.py** - Capture live video from webcam/USB camera
- **image_processing.py** - Basic image processing operations (thresholding, edge detection)

## Purpose
Quick reference examples for learning OpenCV basics before tackling camera calibration and pose estimation.
